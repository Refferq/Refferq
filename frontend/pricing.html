Frequently Asked Questions

Is Refferq really free forever?

Yes! Refferq is 100% open source under the MIT license. You can self-host it for free, modify it, and even use it commercially without any restrictions. The self-hosted version has no limitations on affiliates, conversions, or features.

What are the hosting requirements?

Refferq runs on Node.js (18+) and requires PostgreSQL for the database. You can deploy it on Vercel, Railway, Render, or any VPS. Minimum recommended specs: 512MB RAM, 1 CPU core. For high traffic, we recommend 2GB+ RAM.

Can I customize the platform?

Absolutely! Refferq is built with Next.js, TypeScript, and Tailwind CSS. You have full access to the source code and can customize every aspect - UI, branding, features, workflows. It's designed to be developer-friendly and extensible.

What support is available?

For self-hosted users, we provide community support via GitHub Discussions and our Discord server. Comprehensive documentation and setup guides are available. For priority support, database backups, and managed hosting, check out our Managed plan.

How does tracking work?

Refferq includes a lightweight JavaScript tracking library (refferq-tracker.js) that you add to your website. It automatically detects referral codes from URL parameters, stores them in cookies/localStorage, and tracks conversions via API calls. It's GDPR-friendly and privacy-focused.

Can I migrate from another platform?

Yes! Refferq provides import tools and scripts to help migrate your existing affiliate data. Our API makes it easy to bulk import affiliates, referrals, and historical data. We also have migration guides for popular platforms in our documentation.
