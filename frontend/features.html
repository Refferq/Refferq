Powerful features to track referrals, manage affiliates, and grow your business.

Affiliate Management

Complete control over your affiliate network

👥

User Management

Manage affiliates with approval workflows. Set status (Pending, Active, Inactive, Suspended) and control access levels.

📊

Performance Tracking

Real-time analytics for each affiliate. Track clicks, conversions, earnings, and conversion rates in one dashboard.

🔗

Custom Referral Codes

Unique referral codes for every affiliate. Auto-generate or customize codes based on affiliate names or preferences.

Advanced Tracking

Know exactly where your conversions come from

🎯

JavaScript Tracking

Lightweight tracking script (refferq-tracker.js) that works on any website. Install once and track forever.

🔐

API Key Management

Secure API keys with public (pk_) and private (sk_) formats. Generate, rotate, and manage keys from your dashboard.

📈

Real-time Analytics

Track referral clicks, conversions, and revenue in real-time. Filter by date range, affiliate, or campaign.

🌍

Geographic Data

See where your traffic comes from. Track IP addresses, countries, and referrer sources for better targeting.

🔄

Webhook Integration

Real-time webhooks for conversions, payouts, and status changes. Integrate with your existing systems seamlessly.

📱

Cross-device Tracking

Track users across devices with persistent cookies and localStorage. Never lose attribution.

Flexible Commissions

Pay your affiliates exactly how you want

💰

Custom Commission Rules

Set percentage-based or fixed commissions. Configure different rates for different products or tiers.

⏰

Recurring Commissions

Reward affiliates for subscription referrals. Set up recurring payouts for ongoing value.

🎁

Performance Bonuses

Create milestone bonuses to incentivize top performers. Automatically award bonuses based on targets.

📊

Balance Management

Track affiliate balances, pending payouts, and payment history. Export reports for accounting.

Communication Tools

Keep your affiliates engaged and informed

📧

Automated Emails

Welcome emails, approval notifications, and payout alerts sent automatically via Resend integration.

📢

Announcements

Send updates, promotions, and news to all affiliates or specific segments directly from your dashboard.

💬

Support Portal

Built-in support system for affiliates to ask questions and get help. Track tickets and response times.

Developer Friendly

Built for developers, by developers

⚡

Next.js 15

Built with the latest Next.js, React 19, and TypeScript for maximum performance and type safety.

🗄️

PostgreSQL + Prisma

Robust database architecture with Prisma ORM. Easy migrations, type-safe queries, and scalability.

🔒

JWT Authentication

Secure authentication with JWT tokens and OTP-based passwordless login via email.

🎨

Tailwind CSS

Beautiful, responsive UI built with Tailwind CSS. Easy to customize and extend.

📚

RESTful API

38+ API endpoints for complete programmatic control. Comprehensive documentation included.

One-click Deploy
